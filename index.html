#Why > Background

What Developers are Saying

I watched Kamil spend 40 minutes with a developer in San Francisco... and he still couldn't write and run a test app on the Gear S.
- SDC 2014, 삼성전자 수석 H
2.3.1 SDK를 설치하다가 다운받는데 시간이 너무 많이 걸려 포기

- Tizen 앱 개발사 AppPoster, Tizen Forum

We have a major barrier of entry to go from download to "I can write+run+test a sample app".
#Why > What should we do

1. Fix current SDK

  • Download as bundle and support CDN
  • Automate the way to run with Samsung product device
  • Listen to users
But what if other developers are saying
  • 윈도우 계정이 "설치해 볼까"인데 SDK가 깔리지 않아요
  • Tizen 앱을 개발하려고 크롬북을 직구했습니다 (응?)
  • 인트라넷에서 공동 개발을 하고 싶어요
  • 삼성 TV가 Tizen 기반이라던데 제가 사용했던 SDK로 개발하면 되죠?
#Why > What should we do

2. Create new SDK on new platform

  • Web-based
    • Cross-platform support
    • Dog-fooding for HTML5 as application platform
  • Server-based
    • Daily-updated
    • One SDK for all Tizen platforms
    • User behavior focused

#Develop > Features

Features

  • Appplication supported: Tizen web app and HTML5 web app
  • Project Wizard, Import/Export Wizard
  • Manifest Editor
  • Content Assist for Tizen Device API, Privilege Checker
  • Packaging (supporting Samsung Account)
  • Launch with Web Simulator and Tizen target
  • Multi-platform: Mobile 2.4.0/2.3.1/2.3.0, Wearable 2.3.1/2.3.0
  • Server Monitoring, DIBS Deployment
